### 1.1 JavaScript nedir? Nerede çalışır?

**JavaScript**, web’in ve modern uygulamaların ana programlama dilidir.

Başta sadece web sayfalarına etkileşim eklemek için çıkmıştır, bugün hem **frontend** hem **backend** tarafında kullanılır.

---

### JavaScript nerede çalışır?

**1) Tarayıcı (Browser)**

- Chrome, Firefox, Safari gibi tarayıcıların içinde çalışır.
- Kullanıcıyla direkt etkileşim kurar. (Yani kullanıcının yaptığı tıklama, yazı yazma, form gönderme, mouse hareketi gibi aksiyonlara anında tepki verir.)
- DOM’a erişir (Sayfadaki butonları, input alanlarını, yazıları ve HTML elemanlarını JavaScript ile okuma, değiştirme veya yeni eleman ekleme imkânı verir)

**Örnek kullanım:**

- Butona tıklanınca işlem yapmak
- Form doğrulama
- Animasyon, popup, modal

> Tarayıcıda çalışan JS = kullanıcı arayüzü odaklı
> 

---

**2) Node.js (Server tarafı)**

- JavaScript’i tarayıcı dışında çalıştırır.
- Backend, API, server uygulamaları yazılır.
- Dosya sistemi, veritabanı, network erişimi vardır.

**Örnek kullanım:**

- REST API yazmak *(web sitesi (tarayıcı), mobil uygulama gibi istemcilerin HTTP üzerinden veri alıp göndermesini sağlayan backend servisleri oluşturmak)*
- Veritabanı işlemleri
- Cron job, background task

> Node.js’te çalışan JS = iş mantığı ve veri odaklı
> 

---

### Tarayıcı vs Node.js farkı (kısa)

- Tarayıcı → UI, DOM, kullanıcı etkileşimi
- Node.js → Server, API, dosya & veri işlemleri
- Ortak dil JavaScript, ortam farklı

---

### ECMAScript nedir?

**ECMAScript (ES)**, JavaScript’in **resmi standart tanımıdır**.

Yani:

- JavaScript’in **nasıl yazılacağını**
- Hangi özelliklerin **var olduğunu**
- Yeni özelliklerin **ne zaman ve nasıl ekleneceğini**

tanımlar.

- JavaScript = Uygulama dili
- ECMAScript = O dilin kuralları ve özellikleri

Örnek:

- `let`, `const` → ES6
- arrow function → ES6
- `async / await` → ES2017

---

### Standardizasyon neden önemli?

- Her tarayıcı ve Node.js aynı kuralları uygular
- Kodlar her ortamda tutarlı çalışır
- Yeni özellikler kontrollü ve geriye uyumlu gelir (JavaScript’e eklenen yeni özellikler, eski yazılmış kodları bozmaz; yıllar önce yazılan kodlar bugün hâlâ çalışmaya devam eder)

> “JavaScript öğreniyorum” demek, aslında
> 
> 
> **ECMAScript standardını öğreniyorum** demektir.
> 

---

### Kısa özet

- JavaScript → Programlama dili
- Tarayıcı → UI & kullanıcı etkileşimi
- Node.js → Server & backend
- ECMAScript → JavaScript’in resmi standardı
